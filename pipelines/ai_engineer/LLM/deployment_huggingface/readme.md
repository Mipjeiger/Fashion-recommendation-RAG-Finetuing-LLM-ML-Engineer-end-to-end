# 👗 Fashion AI Assistant

A RAG-powered fashion chatbot built with:

- 🦜 **LangChain** — RAG chain orchestration
- 🤗 **HuggingFace** — Embeddings + LLM inference
- 🗄️ **Qdrant Cloud** — Vector storage
- 🎨 **Gradio** — Chat UI

## Features
- Product recommendations from catalog
- PDF report insights integration
- Semantic similarity search
- Zero local compute — fully cloud-based
